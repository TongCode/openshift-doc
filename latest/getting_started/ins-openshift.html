安装Openshift Origin

  • 获取openshift的二进制包
  • 解压并配置环境
  • 启动openshift服务
  • 参考文档

获取openshift的二进制包

首先我们需要从 openshift的git仓库 中下载可执行文件，这里我们主要使用的是V1.3.0版本。

图 1. git发布包列表
这里我们主要下载openshift-origin-server的发布包，其他关于Client的程序包，我们可以依据自己的工作环境来自行下载。

解压并配置环境

下载完成后，解压到任意目录中，可以看到openshift的运行文件

图 2. openshift文件列表

解压完后需要将当前的目录添加到环境变量中，方便我们可以调用openshift脚本。配置完成后执行命令：

$ oc version

可以看到相应的版本信息输出。

图 3. openshift版本信息

启动openshift服务

我们先以All-In-One的方式进行启动，该模式下master和node将会在本机运行，如果需要以多node的方式运行，后续会进行相关介绍。 执行命令：

$ ./openshift start

将会看到openshift的启动日志信息，启动完成后打开浏览器输入openshift的服务地址：https://服务器IP:8443/console 来访问。

图 4. openshift登陆界面

也可以以客户端的形式访问，执行命令：

$ oc login

第一次执行可能会需要确认master的服务地址，默认为localhost:8443，我们可以修改为自己的服务地址，如果是本机直接默认回车就行。然后安装提示 输入username和password，就可以完成登陆。

图 5. openshift Client登陆

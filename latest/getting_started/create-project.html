创建Projet

  • 使用UI创建project
  • 使用命令行创建项目

在openshift中，一个用户登录平台的账户下，可以创建多个project，每个project是可以当做namespace来使用。

使用UI创建project

首先我们需要通过浏览器来登录到Web控制台。

访问的地址为https://master服务的ip:8443。

点击 New Project 按钮，在打开的表单中填写project的相关信息。

填写完成后点击 Create 完成Project的创建，创建完成后默认会进入到 Add to Project 页面。

我们可以通过页面上方的导航进入到项目信息界面。

我们还可以通过，页面左上角的 Home 图标，进入到当前账户下的项目列表页面。

使用命令行创建项目

openshift平台自带的 oc Client工具也可以帮助我们完成项目的创建等其他操作，接下来我们将使用 terminal 中使用Client工具来完成项目创建操作。

首先我们需要使用 Client 工具登陆到平台中，具体的操作可以参照登陆openshift平台。 登陆成功后可以执行命令：

$ oc new-project -h (1)
1 -h 的参数，适用于所有的openshift命令，意思是获取帮助，可以输出命令的详细介绍。

通过查看帮助信息我们可以知道在创建的时候可以使用的参数信息，执行创建Project的命令：

$ oc new-project first-test-cli --display-name=通过命令创建的项目 --description=命令行创建项目的说明 (1) (2) (3)
1 first-test-cli 指定的Project Name。
2 --display-name 项目显示的名称。
3 --description 项目的介绍信息。

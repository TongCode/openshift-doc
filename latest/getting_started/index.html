openshift是一个基于xxx的开源项目，主要为服务提供弹性伸缩、动态扩展等功能。

安装 Docker

我们将要在环境中的所有机器上安装Docker，批量安装的方法可以使用Ansible、Puppet等工具。这里我们使用手动安装方式，主要使用官方的安装脚本进行安装。

在线脚本安装

Docker官方给我们提供了安装的脚本，我们可以从 get.docker.com 来获取安装脚本，或者执行命令：

$ wget -qO- https://get.docker.com/ | sh

脚本安装

有时可能因为服务器响应关系，导致在线脚本获取一直失败，我们可以通过已经下载好的脚本来进行安装，首先我们 下载脚本，放入到任意目录，然后执行脚本：

$ ./docker-ins.sh

或者我们可以通过网上教程来安装Docker。

安装完成后，我们需要检测下是否安装成功，执行命令：

$ docker --version

来查看是否有Docker的版本信息显示。
